VISION

My vision is a Quarto poster template customized for VCU Capstone projects, easily customizable for other VCU-related activities. This poster template would use the VCU Brand Standard fonts and colors, and implement a pleasing and easily modifiable format, printed directly to PDF without the need for google slides, powerpoint, or figma.

The user creates a QMD file using a text editor (VS code?), modifies the YAML header, then adds their content to the QMD. Stylings, layout and formatting is left to the template styles.

When the QMD is rendered, Quarto would create a PDF of the appropriate size, perfect for printing on large poster paper and mounting on poster board. No powerpoint, no google slides, no interim files, no mouse surgery. Because it’s Quarto, formulas, computations, images and all the goodies can be embed into the single Quarto QMD file. AND, it would all live cleanly within a github repo.

If we build this template, I envision the YAML header below would be used to create the final pumphouse poster (above).

Preferred approach

I believe that the best way to accomplish this project is to create a new typst extension format. Below are instructions for getting started.

I envision the initial extension named “poster-vcu”. Future extensions could be customized for student projects, faculty research labs, department posters, etc.

How to help

Following this instructions above, I created a new typst extension called “poster-vcu”. The code is stored in this repo at this location:

https://github.com/vcu-ssg/ssg-poster-library/tree/main/typst-templates/poster-vcu/poster-vcu

It’s already for someone to start building. FORK this repo and create your own! Then, if you want to share, issue a pull request for me to bring your changes back into this repo.
